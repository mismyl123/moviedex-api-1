# MOVIEDEX-API
***by Richard Folea***  Partnered with Michelle LaLonde 10/20/20
___
##### Example cURL:
```bash
curl -X GET 'http://localhost:9999/movie?genre=Drama&country=United+States&avg_vote=1' -H "Authorization: Bearer YOUR-API-KEY"
```
